# MSCS532_Assignment6

## Repository Contents
- `selection_algorithms.py` - source code for deterministic and randomized selection
- `benchmark_selection.py` - benchmarking script for empirical analysis
- `arrays.py` - dynamic array operations
- `stacks_queues.py` - stack and queue implementations using arrays and linked lists
- `linked_lists.py` - singly linked list and optional rooted tree
- `dataStructures_demo.py` - driver file to test all structures
- `report.md` - written report for both parts of the assignment

## How to Run
1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open a terminal in the project folder.
4. Run:
   ```bash
   python main_demo.py
   ```
5. Run:
   ```bash
   benchmark_selection.py
   ```
